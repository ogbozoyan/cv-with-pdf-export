# CV

This is a code bundle for CV with PDF Export. The original project is available at https://www.figma.com/design/zXPqHK1nzQISEIbPMKQa7J/Stylish-CV-with-PDF-Export.

## Requirements

- Node.js `>=18` (recommended: `20`, see `.nvmrc`)

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
